# CubeLab 3×3 · Three.js + Vue 3

一个可直接启动的经典三阶魔方项目。使用 Vue 3 管理交互界面，使用 Three.js 渲染 27 个独立小方块，并通过旋转组实现连续的 90° / 180° 动画。

## 功能

- 真实的 3×3×3 魔方结构：27 个独立小方块、六面贴纸、灯光与阴影。
- 鼠标拖拽旋转观察视角，滚轮缩放。
- 点击任意彩色贴纸，转动对应一层；按住 `Shift` 点击则反向转动。
- 支持 `R L U D F B`、逆转（如 `R'`）和双转（如 `R2`）。
- 按钮控制、键盘控制、公式输入播放。
- 随机打乱、计时、操作步数、撤销、重做、自动回放复原、重置。
- 响应式页面，适配桌面和手机浏览器。

## 启动

要求：Node.js 20+。

```bash
npm install
npm run dev
```

打开命令行输出的本地地址，通常为：`http://localhost:5173`。

## 构建生产包

```bash
npm run build
npm run preview
```

## 自检

```bash
npm run check
```

该命令会验证六面四次循环复原、公式与逆公式回退、以及长序列中 27 个小方块的逻辑坐标唯一性。

构建产物位于 `dist/`。

## 操作说明

| 操作 | 说明 |
| --- | --- |
| 鼠标左键拖拽 | 旋转观察视角 |
| 鼠标滚轮 | 缩放 |
| 点击彩色贴纸 | 顺时针转动该面所在层 |
| Shift + 点击彩色贴纸 | 逆时针转动 |
| `R L U D F B` | 执行对应层转动 |
| `Shift + R/L/U/D/F/B` | 执行对应逆转 |
| 公式输入框 | 输入空格分隔的公式，例如 `R U R' U'` |

## 设计要点

每次转动会把对应 9 个小方块临时挂到同一个 `THREE.Group`，通过该组的四元数执行缓动旋转。动画结束后，程序会将小方块重新挂回根节点，并把逻辑网格坐标、四元数朝向归一化到准确的整数位置，从而保证长期多次操作也不会产生浮点误差累积。
